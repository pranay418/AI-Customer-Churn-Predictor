# AI Customer Churn Predictor

## Abstract

AI Customer Churn Predictor is an end-to-end machine learning web application that predicts customer churn for SaaS and subscription-based businesses. The system identifies high-risk customers, analyzes churn factors, and provides personalized retention recommendations through an interactive dashboard to support data-driven business decisions.
